斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
给定 n ，请计算 F(n) 。

### 确定dp数组（dp table）以及下标的含义

dp[i] 为 第i个斐波那契数



### 确定递推公式

dp[i] = dp[i-1] + dp[i-2]



### dp数组如何初始化

dp[0] = 0

dp[1] = 1



### 确定遍历顺序

一维遍历，由2遍历至n-1



### 举例推导dp数组

0 1 1 2 3 5 8 13 



```Java
    public int fib(int n) {
        if(n == 0) return 0;
        if(n == 1) return 1;

        int[] dp = new int[32];
        dp[0] = 0;
        dp[1] = 1;
        for(int i = 2;i <= n;i++){
            dp[i] = dp[i-1] +dp[i-2];
        }

        return dp[n];
    }
```
